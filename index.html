    Web Community para usuários de Software Livre.

    Project Name / Project History

    Freenõs pode ser uma rede, uma comunidade uma IDEIA. Consiste em levar o OpenSource e o Software Livre para todos que tenham interesse, ou que queiram se aventurar nesse ótimo mundo. Para usar, basta começar a editar o código, dar ideias, sugestões.

    Installation

    Primeiro baixa todo o código contido neste git: https://github.com/pedrox86lopes/ProjetoFreenos

    Após o download, rode em um LocalServer; caso necessite de permissões $ chmod 777 na pasta.
